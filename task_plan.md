# task_plan

## 需求清单
- 净值/结余曲线模块拆分：将日/月/年/全部收益统计表及以下部分拆分为新模块“收益统计表”。
- 账户统计-账户总览：持仓盈亏与持仓收益率统计纳入库存费。
- 账户统计-当前持仓：顶部持仓盈亏与持仓收益率纳入库存费。
- 账户统计-当前持仓-按产品汇总：最后一条盈亏金额加上对应产品库存费。
- 账户统计-当前持仓-持仓明细：未展开状态最右侧金额加上对应交易库存费。
- 账户统计-交易记录：各历史记录未展开状态最右侧金额加上对应交易库存费。
- 账户统计-交易记录底部文案顺序改为：交易次数、盈亏、库存费、结余。
- 账户统计=交易统计：毛利、亏损金额加上库存费。

## 影响范围（待定位）
- 账户统计页面 UI 与数据聚合
- 净值/结余曲线模块与收益统计表模块

## 验证点（待补充）

- 顶部持仓盈亏/收益率口径一致
- 交易统计毛利/亏损包含库存费
- 文案顺序更新
