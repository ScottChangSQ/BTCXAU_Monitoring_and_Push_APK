# Vintage UI Redesign（60s–80s）

## 1. 设计目标
- 保留全部原有交互、信息结构、数据字段与页面跳转。
- 将视觉从“现代深色科技风”切换到“复古海报/杂志感”。
- 氛围控制：温暖、怀旧、轻做旧，不走霓虹或强科技动画。

## 2. 视觉方向（第一性原则拆解）
| 需求 | 设计决策 | 落地方式 |
|---|---|---|
| 怀旧、温暖 | 用纸张色背景 + 木质墨色文字 | `bg_surface/bg_card/text_primary` |
| 经典不过时 | 控制饱和度，避免高亮荧光 | 暖棕/暗红/橄榄/复古蓝 |
| 报刊层级 | 标题衬线，正文高可读 sans | 章节标题 `serif`，正文 `sans-serif` |
| 轻微做旧 | 细边框、低圆角、微渐变颗粒感 | `bg_app`/卡片/按钮 layer + stroke |

## 3. 设计系统（Design System）

### 3.1 颜色规范
| Token | 色值 | 用途 |
|---|---|---|
| `bg_primary` | `#2F241C` | 深色文字/深色强调前景 |
| `bg_surface` | `#E7D7C0` | 全局页面底色（纸面） |
| `bg_card` | `#F1E6D4` | 卡片底色 |
| `bg_input` | `#E1D0B3` | 输入区、未选中控件底色 |
| `stroke_card` | `#9F825F` | 统一描边 |
| `text_primary` | `#3A2B1F` | 主文本 |
| `text_secondary` | `#776149` | 次级文本 |
| `accent_gold` | `#C79A3A` | 主强调（选中标签/按钮） |
| `accent_blue` | `#6D88A8` | 次强调（品种选中） |
| `accent_red` | `#9B4A3C` | 负向/风险 |
| `accent_green` | `#59693D` | 正向/收益 |
| `accent_cyan` | `#4F6F84` | 图表辅助强调 |
| `vintage_orange` | `#BE6C3A` | 按压态/交互反馈 |

### 3.2 字体规范
| 层级 | 字体 | 说明 |
|---|---|---|
| 页面/模块标题 | `serif` | 强化复古报刊感（如“账户总览/交易记录”） |
| 正文/数据/控件 | `sans-serif` | 保证数字和交易信息可读性 |

### 3.3 组件规范
| 组件 | 规范 |
|---|---|
| 卡片 | 浅纸张底、1dp 描边、6dp 低圆角、轻微阴影 |
| 按钮 | 4dp 小圆角，默认浅底描边，按下橙色反馈 |
| Chip/Tab | 选中：芥末黄底 + 深色字；未选中：浅棕底 + 描边 |
| 表格 | 表头深一阶纸色、数据行浅纸色、统一细分隔 |
| 下拉框 | 复古浅底 + 深色箭头，弹层同纸张卡片体系 |
| 日志等级Tag | 暖色半透明底 + 边框（信息/警告/错误） |

### 3.4 动效规范
- 保持系统默认过渡，不增加科技感强动画。
- 状态变化以颜色和明暗反馈为主（轻、快、克制）。

## 4. 关键页面示例

### 4.1 首页（Market Monitor）
- 结构不变：连接状态、监控状态、行情卡、记录卡、阈值卡。
- 视觉变化：
  - 标题改衬线（如“市场卡片标题/最近记录/阈值设置”）。
  - 按钮与切换改为复古按钮体系（小圆角+细边）。
  - 背景使用纸张渐变与轻颗粒层，降低“电子屏”观感。

### 4.2 列表页（Log）
- 结构不变：返回、全选、删除、日志列表。
- 视觉变化：
  - 页标题改衬线大字，接近杂志栏头。
  - 日志项卡片采用纸张色卡 + 等级徽章暖色块。
  - 文本主次对比调整为“深墨色 + 棕灰副文案”。

### 4.3 详情页（Account Stats）
- 结构不变：账户总览、曲线、收益统计表、当前持仓、交易记录、交易统计。
- 视觉变化：
  - 各章节标题统一衬线风格，形成清晰分栏感。
  - 表格、筛选器、条目卡片统一复古边框与背景层级。
  - 图表继续保留正负色语义，但色相切换为复古调。

## 5. 已落地资源范围
- 全局：`values/colors.xml`、`values/themes.xml`
- 组件皮肤：`drawable/bg_*.xml`（卡片、按钮、chip、spinner、表格、浮窗）
- 关键页面：
  - `layout/activity_main.xml`
  - `layout/activity_log.xml`
  - `layout/activity_account_stats.xml`
  - `layout/activity_settings.xml`
- 列表与筛选条目：`item_spinner_filter*.xml` 等

## 6. 兼容与功能保证
- 未修改页面 ID、事件绑定、ViewModel/Adapter 数据流。
- 所有业务逻辑（监控、账户、统计、筛选、展开/收起）保持原行为。
- 本次变更是“样式重构”，不是功能重构。
